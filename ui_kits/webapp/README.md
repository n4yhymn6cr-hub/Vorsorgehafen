# Vorsorgehafen Web App UI Kit

## Overview
High-fidelity, click-through prototype of the Vorsorgehafen web application. Covers the core user journey from onboarding through the dashboard.

## Screens
1. **Welcome / Landing** — first impression, CTA to start
2. **Onboarding** — 4-step questionnaire (Situation → Ziele → Einkommen → Ergebnis)
3. **Dashboard** — Vorsorge-Übersicht, Sparrate, Lückenanalyse
4. **Plan Detail** — Einzelne Vorsorge-Bausteine

## Design Width
`1280px` — responsive down to 960px

## Components
- `Components.jsx` — Sidebar, Nav, Badge, Card, Button, ProgressBar, Input, StatCard
- `Dashboard.jsx` — Dashboard screen
- `Onboarding.jsx` — Onboarding flow (multi-step)

## Fonts
- Cormorant Garamond (Google Fonts) — Display/Headlines
- DM Sans (Google Fonts) — Body/UI
- DM Mono (Google Fonts) — Numbers/Values

## Icon Set
Lucide Icons via CDN (`https://unpkg.com/lucide@latest`)

## Notes
- Not production code — cosmetic recreations only
- Click-through navigation between screens
- No real data or API connections
